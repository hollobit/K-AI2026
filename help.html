AND 검색

공백 또는 쉼표로 키워드를 구분하면 AND 조건으로 검색됩니다.

예: 반도체 인프라 → "반도체"와 "인프라" 모두 포함된 결과만 표시

  • 과제별 상세 탭: 매칭 과제 카드 + 하단에 글로벌 사례 카드
  • 검색 카운트 바: "과제 N건 · 참고 N건 · 각주 N건 · 글로벌 사례 N건"
  • 검색어 하이라이팅: 매칭 키워드가 노란색으로 강조
  • 자동완성: 입력 시 관련 과제명·키워드 자동 제안
  • 최근 검색어: localStorage에 저장, 검색창 포커스 시 표시
💡 팁: 검색 후 각 탭으로 이동하면 검색어가 자동으로 해당 탭의 필터에 동기화됩니다.

6. 분석 기능

탭 7: 유사/중복 분석 — TF-IDF 코사인 유사도

  • 알고리즘: 과제 텍스트에서 불용어 제거 → TF-IDF 벡터 생성 → 코사인 유사도 계산
  • 보정: 같은 전략분야(+15%), 공통 부처(+10%)
  • 임계값: 슬라이더(30~90%)로 정밀도 조절
  • 보기 모드: 쌍별 보기(과제 A↔B 비교) / 과제별 보기(한 과제의 모든 유사 과제)
  • 비동기 계산: 진행률 바 표시, UI 블로킹 방지

탭 8: 연관 과제 — Force-directed 네트워크 그래프

  • 연관도 산정: 직접언급(50) + 참고자료(40) + 키워드(20) + 부처(20) + 전략분야(20) = 최대 150점 → 0~100점 정규화
  • 관계 분류: 선행·병행·후행 (기한 비교 + 정책축/전략분야 순서 기반)
  • 물리 시뮬레이션: 500프레임 자동 정지, 노드 드래그 가능
  • 뷰 모드: 에고 네트워크(특정 과제 중심) / 전체 조감도

탭 10: 표준화 현황

  • 5개 유형: 법·제도 정비 / 기술 표준 / 안전·신뢰 기준 / 가이드라인·지침 / 평가·인증 체계
  • 전략분야 5(AI 규제혁신) 전체 과제 자동 포함
  • 키워드 기반 필터링: 표준, KS, ISO, ITU, 인증, 안전 등

Q. 타임라인에서 같은 과제가 여러 분기에 나타나는 이유는?▼

Q. 인터넷 연결 없이도 사용할 수 있나요?▼

거의 가능합니다. 모든 데이터가 HTML 파일 내에 인라인으로 포함되어 있어 외부 서버 요청이 없습니다.

다만 구글 폰트(Noto Sans KR)와 Chart.js CDN은 인터넷이 필요합니다. 폰트가 로드되지 않으면 시스템 기본 폰트로 대체됩니다.

Q. 화면을 인쇄하거나 PDF로 저장할 수 있나요?▼

헤더의 "내보내기" 버튼을 클릭하면 현재 화면을 PNG 또는 PDF로 저장할 수 있습니다.

브라우저의 인쇄 기능(Ctrl+P)도 사용 가능하며, 인쇄 최적화 CSS가 적용되어 있습니다.

Q. 다크 모드는 어떻게 전환하나요?▼

헤더 우측의 🌙/☀️ 버튼을 클릭하면 라이트/다크 모드가 전환됩니다. 설정은 localStorage에 저장되어 재방문 시에도 유지됩니다.

OS의 다크 모드 설정이 켜져 있으면 첫 방문 시 자동으로 다크 모드가 적용됩니다.

Q. 키보드 단축키가 있나요?▼

네, 다음 단축키를 지원합니다:

• Ctrl+F — 전역 검색창 포커스

• 1~9, 0 — 탭 1~10 전환 (종합현황판~표준화현황)

• ←/→ — 모달 내 이전/다음 과제 이동

• Esc — 모달 닫기, 검색 초기화

Q. 페이지 로딩이 느려요▼
